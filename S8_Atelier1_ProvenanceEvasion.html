🎯 Objectif : Avant d'ajouter quoi que ce soit, s'assurer que la maquette S7 est pleinement fonctionnelle : routage inter-sites opérationnel, DNS résolvant les noms, et VLAN 15 (Serveurs) accessible depuis tous les sites.

Vérification du VLAN 15 (Serveurs)

Depuis PC-Personnel-Apt (VLAN 11), ping vers le serveur DNS/DHCP (192.168.15.3). Ce ping doit réussir car le routeur d'Apt assure le routage entre VLAN 11 et VLAN 15.

Phase B — Configuration de NAS-APT (siège d'Apt) ~50 min

3
Ajouter le serveur NAS-APT dans la topologie

Dans la zone du siège d'Apt, ajoute un serveur générique (Server-PT). Nomme-le NAS-APT. Branche-le sur le switch d'Apt via un câble droit. Affecte le port au VLAN 15.

Switch-Apt(config)# interface Switch-Apt(config-if)# Switch-Apt(config-if)# Switch-Apt(config-if)# no shutdown Switch-Apt(config-if)# exit Switch-Apt(config)# end Switch-Apt# write memory

Phase C — Configuration de NAS-Roussillon (site secondaire) ~40 min

🎯 Objectif : Axentis a prévu un deuxième NAS à Roussillon pour stocker les données locales de ce site (réservations Roussillon, plannings locaux). Il doit être accessible depuis Roussillon mais aussi depuis le siège d'Apt.
6
Ajouter NAS-Roussillon dans la topologie

Dans la zone du site de Roussillon, ajoute un serveur générique. Nomme-le NAS-Roussillon. Il se place dans le VLAN 25 (Serveurs Roussillon) avec l'IP 192.168.25.4.

Switch-Roussillon(config)# Switch-Roussillon(config-vlan)# name Switch-Roussillon(config-vlan)# exit

Configurer le service FTP de NAS-Roussillon

Active le service FTP sur NAS-Roussillon avec les mêmes 3 comptes que NAS-APT (mêmes identifiants et droits). Crée ensuite un fichier test : dans un PC de Roussillon, crée un fichier texte nommé planning-roussillon.txt et envoie-le sur NAS-Roussillon.

─── Depuis PC-Personnel-Roussillon (192.168.21.x) ─── PC> ftp 192.168.25.4 ← NAS local de Roussillon Username: employe Password: ciel2026 ftp> put planning-roussillon.txt ftp> dir ← vérifie la présence du fichier ftp> quit
