2. Configuration du port Trunk sur le switch

2.1 Qu'est-ce qu'un trunk — rappel appliqué

Le port Gi0/1 du switch relie le switch au routeur. Il doit transporter les 4 VLANs du site (21, 22, 23, 24) sur un seul câble physique. Pour cela, on le passe en mode Trunk avec encapsulation 802.1Q.

Schéma Trunk — Switch ↔ Routeur (site Roussillon)
SWITCH (Gi0/1 Trunk) ═══════════[câble 802.1Q]═══════════ ROUTEUR (Gi0/0) │ ├── VLAN 21 (Personnel) → Gi0/0.21 → 192.168.21.1 ├── VLAN 22 (WiFi Clients) → Gi0/0.22 → 192.168.22.1 ├── VLAN 23 (Caméras) → Gi0/0.23 → 192.168.23.1 └── VLAN 24 (Admin) → Gi0/0.24 → 192.168.24.1 Chaque trame est étiquetée avec son VID 802.1Q sur le câble trunk. VLAN natif : 99 (inutilisé) — sécurité contre le VLAN hopping.
Étape 1 Sur le switch Cisco 2960, configurez le port GigabitEthernet0/1 en mode Trunk. Autorisez uniquement les VLANs 21, 22, 23 et 24. Définissez le VLAN natif sur 99 (créez-le d'abord !).
📝 À faire — Port Trunk sur le switch

3. Configuration du Router-on-a-Stick

3.1 Activation de l'interface physique du routeur

Sur le routeur Cisco 1941, l'interface physique GigabitEthernet0/0 (reliée au switch) doit être activée sans adresse IP — ce sont les sous-interfaces qui porteront les adresses. On active d'abord le "chassis" physique.

Étape 2 Sur le routeur, passez en mode de configuration. Accédez à l'interface Gi0/0 et activez-la (no shutdown). Ne lui attribuez aucune adresse IP.
📝 À faire — Activation Gi0/0

3.2 Création des sous-interfaces 802.1Q

Pour chaque VLAN, on crée une sous-interface sur le routeur. La syntaxe est : interface GigabitEthernet0/0.VID. On y associe l'encapsulation 802.1Q et l'adresse IP de passerelle.

Étape 3 Créez les 4 sous-interfaces (Gi0/0.21, Gi0/0.22, Gi0/0.23, Gi0/0.24) avec les bonnes encapsulations et adresses IP.
📝 À faire — 4 sous-interfaces Router-on-a-Stick
