📍 Contexte : En S5, vous avez interconnecté Apt, Roussillon et Saignon. Il reste à brancher Cucuron et à vérifier que toutes les communications autorisées fonctionnent entre les 4 sites. 2. Packet Tracer — Connexion et configuration de Cucuron

2.1 Vérification de l'état actuel

📋 Consigne : Ouvrez votre fichier Packet Tracer de la séance S5. Avant toute modification, faites une capture de l'état actuel.
Étape 1 Sur le routeur d'Apt, tapez show ip route et notez ci-dessous les routes actuellement présentes.
Étape 2 Tapez show ip interface brief sur le routeur d'Apt. Vérifiez que Serial0/0/2 (liaison vers Cucuron) est bien en état UP/UP. Si elle est administratively down, notez-le.

2.2 Ajout du routeur de Cucuron dans Packet Tracer

📋 Consigne : Si le routeur de Cucuron n'est pas encore présent dans votre topologie, ajoutez-le maintenant. Utilisez un routeur Cisco 1941 avec un module Serial.
Étape 3 Ajoutez le routeur de Cucuron (Cisco 1941). Reliez-le au routeur d'Apt par une liaison Serial (câble DCE/DTE). Vérifiez que l'horloge DCE est bien côté Apt.
Étape 4 Configurez l'interface Serial0/0/2 du routeur d'Apt :
Configuration de la liaison WAN Apt—Cucuron (sur Routeur-Apt) Routeur-Apt# configure terminal Routeur-Apt(config)# interface Serial0/0/2 Routeur-Apt(config-if)# ip address 10.0.2.1 255.255.255.252 Routeur-Apt(config-if)# clock rate 64000 ← DCE uniquement, côté Apt Routeur-Apt(config-if)# no shutdown Routeur-Apt(config-if)# description WAN-vers-Cucuron Routeur-Apt(config-if)# end Routeur-Apt# write memory
Étape 5 Configurez l'interface Serial0/0/0 du routeur de Cucuron :
Configuration de la liaison WAN Cucuron—Apt (sur Routeur-Cucuron) Routeur-Cucuron# configure terminal Routeur-Cucuron(config)# interface Serial0/0/0 Routeur-Cucuron(config-if)# ip address 10.0.2.2 255.255.255.252 Routeur-Cucuron(config-if)# no shutdown Routeur-Cucuron(config-if)# description WAN-vers-Apt Routeur-Cucuron(config-if)# end
🧠 Vérification étape 5

2.3 Configuration des VLANs et des sous-interfaces de Cucuron

Étape 6 Configurez le routage inter-VLAN (Router-on-a-Stick) sur le routeur de Cucuron. Les VLANs de Cucuron sont : Personnel (VLAN 41 — 192.168.41.0/27), Clients WiFi (VLAN 42 — 192.168.42.0/25), Caméras (VLAN 43 — 192.168.43.0/28), Administration (VLAN 44 — 192.168.44.0/29).
Sous-interfaces VLAN sur Routeur-Cucuron Routeur-Cucuron(config)# interface GigabitEthernet0/0 Routeur-Cucuron(config-if)# no shutdown Routeur-Cucuron(config-if)# exit Routeur-Cucuron(config)# interface GigabitEthernet0/0.41 Routeur-Cucuron(config-subif)# encapsulation dot1Q 41 Routeur-Cucuron(config-subif)# ip address 192.168.41.1 255.255.255.224 Routeur-Cucuron(config-subif)# exit Routeur-Cucuron(config)# interface GigabitEthernet0/0.42 Routeur-Cucuron(config-subif)# encapsulation dot1Q 42 Routeur-Cucuron(config-subif)# ip address 192.168.42.1 255.255.255.128 Routeur-Cucuron(config-subif)# exit Routeur-Cucuron(config)# interface GigabitEthernet0/0.43 Routeur-Cucuron(config-subif)# encapsulation dot1Q 43 Routeur-Cucuron(config-subif)# ip address 192.168.43.1 255.255.255.240 Routeur-Cucuron(config-subif)# exit Routeur-Cucuron(config)# interface GigabitEthernet0/0.44 Routeur-Cucuron(config-subif)# encapsulation dot1Q 44 Routeur-Cucuron(config-subif)# ip address 192.168.44.1 255.255.255.248 Routeur-Cucuron(config-subif)# end Routeur-Cucuron# write memory

2.4 Configuration des routes statiques pour Cucuron

Étape 7 Ajoutez sur le routeur d'Apt la route vers le réseau de Cucuron.
Route vers Cucuron — à ajouter sur Routeur-Apt Routeur-Apt(config)# ip route 192.168.40.0 255.255.252.0 10.0.2.2 ← agrège les 4 VLANs de Cucuron Routeur-Apt(config)# end Routeur-Apt# write memory
Étape 8 Configurez les routes sur le routeur de Cucuron : il doit connaître tous les autres sites ET avoir une route par défaut vers Apt.
⚠ Ne pas oublier ! Vérifiez également que tous les autres routeurs (Roussillon, Saignon) n'ont pas de route statique manquante vers le réseau 192.168.40.0/22 de Cucuron. Si vous avez ajouté une route agrégée 0.0.0.0/0 pointant vers Apt, cette vérification est inutile — Apt se charge du relais.
