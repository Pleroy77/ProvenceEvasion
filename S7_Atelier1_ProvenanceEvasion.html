Contexte de l'atelier

📍 Mission Axentis : Les quatre sites de Provence Évasion sont désormais interconnectés (S5-S6). Laurent Ferracci exige que les techniciens et les serveurs soient joignables par leur nom plutôt que par leur adresse IP. Axentis Informatique vous confie la mise en place du serveur DNS interne, hébergé au siège d'Apt. Dans cet atelier 1, vous allez construire la topologie dans Packet Tracer, configurer le serveur DNS, créer les enregistrements de zone directe, et valider les résolutions depuis deux postes clients (Apt et Roussillon).

2. Construction de la topologie dans Packet Tracer

2.1 Placer les équipements

🛠 Consigne : Ouvrez un nouveau fichier Packet Tracer. Placez les équipements suivants dans l'espace de travail et reliez-les selon le schéma ci-dessus. Pour la liaison WAN, utilisez des câbles séries (DCE côté Routeur-APT).
① Ajouter 2 routeurs (modèle 1941 ou 2811) : Routeur-APT, Routeur-ROUSSILLON
② Ajouter 1 switch (2960) côté Apt uniquement (Switch-APT)
③ Ajouter 1 serveur (Server-PT) : Serveur-DNS
④ Ajouter 2 PC (PC-PT) : PC-APT, PC-ROUSSILLON
⑤ Câbler : PC-APT → Switch-APT (Fa0/1), Serveur-DNS → Switch-APT (Fa0/2), Switch-APT → Routeur-APT Fa0/0 (Fa0/24), Routeur-APT Se0/0/0 → Routeur-ROUSSILLON Se0/0/0 (câble série), Routeur-ROUSSILLON Fa0/0 → PC-ROUSSILLON Fa0
⑥ Sur Routeur-APT, activer l'horloge série : clock rate 64000 sur l'interface Se0/0/0 (côté DCE)

2.2 Configuration IP du Routeur-APT

Le Routeur-APT gère deux sous-réseaux côté LAN (VLAN 11 et VLAN 15). Dans Packet Tracer on peut utiliser directement deux interfaces Fa distinctes ou une interface avec sous-interfaces. Ici, pour simplifier, on utilise deux interfaces Fa distinctes si le routeur en dispose, sinon deux sous-interfaces sur Fa0/0.

Configuration Routeur-APT Router> enable Router# configure terminal Router(config)# hostname Routeur-APT ── Interface LAN vers VLAN 11 (Personnel Apt) ── Routeur-APT(config)# interface FastEthernet0/0 Routeur-APT(config-if)# ip address 192.168.11.254 255.255.255.0 Routeur-APT(config-if)# no shutdown Routeur-APT(config-if)# exit ── Interface LAN vers VLAN 15 (Serveurs Apt) ── Routeur-APT(config)# interface FastEthernet0/1 Routeur-APT(config-if)# ip address 192.168.15.254 255.255.255.0 Routeur-APT(config-if)# no shutdown Routeur-APT(config-if)# exit ── Interface WAN vers Roussillon ── Routeur-APT(config)# interface Serial0/0/0 Routeur-APT(config-if)# ip address 10.0.12.1 255.255.255.252 Routeur-APT(config-if)# clock rate 64000 ← DCE uniquement Routeur-APT(config-if)# no shutdown Routeur-APT(config-if)# exit ── Routes statiques ── Routeur-APT(config)# ip route 192.168.21.0 255.255.255.0 10.0.12.2 Routeur-APT(config)# end Routeur-APT# write memory

2.3 Configuration IP du Routeur-ROUSSILLON

Configuration Routeur-ROUSSILLON Router(config)# hostname Routeur-ROUSSILLON Routeur-ROUSSILLON(config)# interface FastEthernet0/0 Routeur-ROUSSILLON(config-if)# ip address 192.168.21.254 255.255.255.0 Routeur-ROUSSILLON(config-if)# no shutdown Routeur-ROUSSILLON(config-if)# exit Routeur-ROUSSILLON(config)# interface Serial0/0/0 Routeur-ROUSSILLON(config-if)# ip address 10.0.12.2 255.255.255.252 Routeur-ROUSSILLON(config-if)# no shutdown Routeur-ROUSSILLON(config-if)# exit ── Route statique : pour joindre Apt (VLAN 11) et les serveurs (VLAN 15) ── Routeur-ROUSSILLON(config)# ip route 192.168.11.0 255.255.255.0 10.0.12.1 Routeur-ROUSSILLON(config)# ip route 192.168.15.0 255.255.255.0 10.0.12.1 Routeur-ROUSSILLON(config)# end Routeur-ROUSSILLON# write memory

2.4 Configuration IP des postes clients

① PC-APT → Desktop → IP Configuration : IP 192.168.11.10 / Masque 255.255.255.0 / GW 192.168.11.254 / DNS 192.168.15.1
② PC-ROUSSILLON → Desktop → IP Configuration : IP 192.168.21.10 / Masque 255.255.255.0 / GW 192.168.21.254 / DNS 192.168.15.1
③ Serveur-DNS → Desktop → IP Configuration : IP 192.168.15.1 / Masque 255.255.255.0 / GW 192.168.15.254 / DNS 127.0.0.1

3. Configuration du service DNS sur le serveur

3.1 Activation du service DNS

① Cliquer sur Serveur-DNS → onglet Services → cliquer sur DNS dans la liste de gauche
② Mettre l'interrupteur DNS Service sur ON
③ Vous voyez deux champs : Name (nom FQDN) et Address (adresse IP cible), puis un bouton Add
