Étape 2 — Configuration de Router-Saignon

2.1 Interface WAN vers Apt

! WAN vers Apt : 10.0.0.6/30 Router-Saig> enable Router-Saig# configure terminal Router-Saig(config)# interface GigabitEthernet0/0 Router-Saig(config-if)# ip address 10.0.0.6 255.255.255.252 Router-Saig(config-if)# no shutdown Router-Saig(config-if)# description WAN-vers-Apt Router-Saig(config-if)# exit

2.2 Sous-interfaces LAN — VLANs Saignon

! Interface physique trunk vers SW-Saig Router-Saig(config)# interface GigabitEthernet0/1 Router-Saig(config-if)# no shutdown Router-Saig(config-if)# exit ! VLAN 31 – Personnel (192.168.31.0/26) Router-Saig(config)# interface GigabitEthernet0/1.31 Router-Saig(config-subif)# encapsulation dot1Q 31 Router-Saig(config-subif)# ip address 192.168.31.1 255.255.255.192 Router-Saig(config-subif)# exit ! VLAN 32 – Séminaires (192.168.32.0/26) Router-Saig(config)# interface GigabitEthernet0/1.32 Router-Saig(config-subif)# encapsulation dot1Q 32 Router-Saig(config-subif)# ip address 192.168.32.1 255.255.255.192 Router-Saig(config-subif)# exit ! VLAN 33 – Caméras (192.168.33.0/28) Router-Saig(config)# interface GigabitEthernet0/1.33 Router-Saig(config-subif)# encapsulation dot1Q 33 Router-Saig(config-subif)# ip address 192.168.33.1 255.255.255.240 Router-Saig(config-subif)# exit ! VLAN 34 – Admin (192.168.34.0/29) Router-Saig(config)# interface GigabitEthernet0/1.34 Router-Saig(config-subif)# encapsulation dot1Q 34 Router-Saig(config-subif)# ip address 192.168.34.1 255.255.255.248 Router-Saig(config-subif)# exit ! Route par défaut vers Apt Router-Saig(config)# ip route 0.0.0.0 0.0.0.0 10.0.0.5 Router-Saig(config)# end Router-Saig# write memory

Étape 4 — Routes statiques manquantes sur Router-Apt

Router-Apt connaît déjà les 4 VLANs de Roussillon (Atelier 1). Il faut maintenant ajouter les routes vers Saignon (4 VLANs) et Cucuron (4 VLANs) :

! Routes vers Saignon (next-hop : 10.0.0.6) Router-Apt(config)# ip route 192.168.31.0 255.255.255.192 10.0.0.6 Router-Apt(config)# ip route 192.168.32.0 255.255.255.192 10.0.0.6 Router-Apt(config)# ip route 192.168.33.0 255.255.255.240 10.0.0.6 Router-Apt(config)# ip route 192.168.34.0 255.255.255.248 10.0.0.6 ! Routes vers Cucuron (next-hop : 10.0.0.10) Router-Apt(config)# ip route 192.168.41.0 255.255.255.192 10.0.0.10 Router-Apt(config)# ip route 192.168.42.0 255.255.255.128 10.0.0.10 Router-Apt(config)# ip route 192.168.43.0 255.255.255.240 10.0.0.10 Router-Apt(config)# ip route 192.168.44.0 255.255.255.248 10.0.0.10 Router-Apt(config)# end Router-Apt# write memory
