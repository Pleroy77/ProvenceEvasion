📋 Contexte Axentis : Laurent Ferracci (PDG de Provence Évasion) vient de valider le plan d'adressage et la configuration DHCP (S4). Il attend maintenant la première interconnexion des sites. Votre mission ce jour : construire la topologie Packet Tracer complète des 4 sites et établir la communication entre le siège d'Apt et le site de Roussillon.

⏱ Durée : 4h — Sauvegardez votre fichier sous le nom PE_S5A1_[Nom]_[Prenom].pkt en fin de séance.
⚠ Important : Cet atelier repart de zéro. Vous pouvez récupérer votre fichier S4 pour copier les configurations des switches et routeurs déjà faites (sous-interfaces LAN, DHCP), mais la topologie WAN est entièrement nouvelle.

Étape 3 — Configuration de Router-Apt

3.1 Interfaces WAN (Gi0/1, Gi0/2, Gi0/3)

Configurez les 3 interfaces WAN de Router-Apt avec les adresses du plan d'adressage :

! Accès configuration Router-Apt Router-Apt> enable Router-Apt# configure terminal ! Interface WAN vers Roussillon Router-Apt(config)# interface GigabitEthernet0/1 Router-Apt(config-if)# ip address 10.0.0.1 255.255.255.252 Router-Apt(config-if)# no shutdown Router-Apt(config-if)# description WAN-vers-Roussillon Router-Apt(config-if)# exit ! Interface WAN vers Saignon Router-Apt(config)# interface GigabitEthernet0/2 Router-Apt(config-if)# ip address 10.0.0.5 255.255.255.252 Router-Apt(config-if)# no shutdown Router-Apt(config-if)# description WAN-vers-Saignon Router-Apt(config-if)# exit ! Interface WAN vers Cucuron Router-Apt(config)# interface GigabitEthernet0/3 Router-Apt(config-if)# ip address 10.0.0.9 255.255.255.252 Router-Apt(config-if)# no shutdown Router-Apt(config-if)# description WAN-vers-Cucuron Router-Apt(config-if)# exit

3.2 Sous-interfaces LAN de Router-Apt (VLAN 11, 13, 14, 15)

! Activation de l'interface physique (trunk) Router-Apt(config)# interface GigabitEthernet0/0 Router-Apt(config-if)# no shutdown Router-Apt(config-if)# exit ! VLAN 11 – Personnel Router-Apt(config)# interface GigabitEthernet0/0.11 Router-Apt(config-subif)# encapsulation dot1Q 11 Router-Apt(config-subif)# ip address 192.168.11.1 255.255.255.0 Router-Apt(config-subif)# exit ! VLAN 13 – Caméras Router-Apt(config)# interface GigabitEthernet0/0.13 Router-Apt(config-subif)# encapsulation dot1Q 13 Router-Apt(config-subif)# ip address 192.168.13.1 255.255.255.0 Router-Apt(config-subif)# exit ! VLAN 14 – Administration Router-Apt(config)# interface GigabitEthernet0/0.14 Router-Apt(config-subif)# encapsulation dot1Q 14 Router-Apt(config-subif)# ip address 192.168.14.1 255.255.255.0 Router-Apt(config-subif)# exit ! VLAN 15 – Serveurs Router-Apt(config)# interface GigabitEthernet0/0.15 Router-Apt(config-subif)# encapsulation dot1Q 15 Router-Apt(config-subif)# ip address 192.168.15.1 255.255.255.0 Router-Apt(config-subif)# exit

Étape 4 — Configuration de Router-Roussillon

4.1 Interface WAN (Gi0/0)

! Interface WAN vers Apt Router-Rous(config)# interface GigabitEthernet0/0 Router-Rous(config-if)# ip address 10.0.0.2 255.255.255.252 Router-Rous(config-if)# no shutdown Router-Rous(config-if)# description WAN-vers-Apt Router-Rous(config-if)# exit

4.2 Sous-interfaces LAN (VLAN 21, 22, 23, 24)

! Interface physique trunk Router-Rous(config)# interface GigabitEthernet0/1 Router-Rous(config-if)# no shutdown Router-Rous(config-if)# exit ! VLAN 21 – Personnel (192.168.21.0/26) Router-Rous(config)# interface GigabitEthernet0/1.21 Router-Rous(config-subif)# encapsulation dot1Q 21 Router-Rous(config-subif)# ip address 192.168.21.1 255.255.255.192 Router-Rous(config-subif)# exit ! VLAN 22 – Clients WiFi (192.168.22.0/25) Router-Rous(config)# interface GigabitEthernet0/1.22 Router-Rous(config-subif)# encapsulation dot1Q 22 Router-Rous(config-subif)# ip address 192.168.22.1 255.255.255.128 Router-Rous(config-subif)# exit ! VLAN 23 – Caméras (192.168.23.0/28) Router-Rous(config)# interface GigabitEthernet0/1.23 Router-Rous(config-subif)# encapsulation dot1Q 23 Router-Rous(config-subif)# ip address 192.168.23.1 255.255.255.240 Router-Rous(config-subif)# exit ! VLAN 24 – Administration (192.168.24.0/29) Router-Rous(config)# interface GigabitEthernet0/1.24 Router-Rous(config-subif)# encapsulation dot1Q 24 Router-Rous(config-subif)# ip address 192.168.24.1 255.255.255.248 Router-Rous(config-subif)# exit

4.3 Route par défaut sur Router-Roussillon

! Tout le trafic non local passe par Router-Apt Router-Rous(config)# ip route 0.0.0.0 0.0.0.0 10.0.0.1

Étape 5 — Routes statiques sur Router-Apt vers Roussillon

Pour que Router-Apt puisse atteindre les 4 VLANs de Roussillon, configurez les 4 routes statiques suivantes :

! 4 routes statiques vers Roussillon (via 10.0.0.2) Router-Apt(config)# ip route 192.168.21.0 255.255.255.192 10.0.0.2 Router-Apt(config)# ip route 192.168.22.0 255.255.255.128 10.0.0.2 Router-Apt(config)# ip route 192.168.23.0 255.255.255.240 10.0.0.2 Router-Apt(config)# ip route 192.168.24.0 255.255.255.248 10.0.0.2

Étape 6 — Configuration des switches et PCs de test

6.1 SW-Apt — Port trunk et port access VLAN 11

! Lien trunk vers Router-Apt SW-Apt(config)# interface GigabitEthernet0/1 SW-Apt(config-if)# switchport mode trunk SW-Apt(config-if)# exit ! Port access VLAN 11 pour PC-Apt-1 SW-Apt(config)# vlan 11 SW-Apt(config-vlan)# name Personnel-Apt SW-Apt(config-vlan)# exit SW-Apt(config)# interface FastEthernet0/1 SW-Apt(config-if)# switchport mode access SW-Apt(config-if)# switchport access vlan 11 SW-Apt(config-if)# exit

6.2 SW-Rous — Port trunk et port access VLAN 21

! Lien trunk vers Router-Rous SW-Rous(config)# interface GigabitEthernet0/1 SW-Rous(config-if)# switchport mode trunk SW-Rous(config-if)# exit ! Port access VLAN 21 pour PC-Rous-1 SW-Rous(config)# vlan 21 SW-Rous(config-vlan)# name Personnel-Roussillon SW-Rous(config-vlan)# exit SW-Rous(config)# interface FastEthernet0/1 SW-Rous(config-if)# switchport mode access SW-Rous(config-if)# switchport access vlan 21 SW-Rous(config-if)# exit
